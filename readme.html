TurtleGraphics

TurtleGraphics is a turtle graphics made on the top of Swift’s function builder. It allows you to use a Logo-like syntax to create and draw lines in your Swift project.

Inspiration

  • TurtleBuilder: https://github.com/zonble/TurtleBuilder

  • I wanted/needed dynamically sizing graphics

  • Because I’m a glutton for punishment :-)

Requirement

  • Swift 5.10 or above

Installation

You can install TurtleGraphics into your project via Swift Package Manager.
